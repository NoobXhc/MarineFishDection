

# 海洋鱼类智能识别系统

基于YOLOv8深度学习模型和PyQt5现代化界面的海洋鱼类识别系统，提供直观的图片、视频和实时摄像头检测功能。

Python-3.9           PyQt5-5.15.9             YOLOv8-8.0.186        OpenCV-4.8.1 

## 核心特性

### 智能图片检测

- **批量处理** - 支持多张图片连续检测和导航
- **格式兼容** - JPG、PNG、WEBP、BMP、TIFF等主流格式
- **实时显示** - 检测结果即时可视化，带文件名和序号
- **一键保存** - 智能命名保存检测结果

### 视频流分析

- **全格式支持** - MP4、AVI、MKV、MOV、WMV等
- **进度控制** - 实时进度条，支持暂停/继续
- **帧抓拍** - 任意时刻保存高质量帧
- **视频导出** - 完整录制检测过程视频

### 实时摄像头

- **即开即用** - 自动检测摄像头设备
- **性能监控** - 实时FPS显示和统计
- **智能录制** - 带时间戳的自动视频录制
- **即时捕获** - 高质量帧抓拍功能

### 智能设置

- **灵敏度调节** - 置信度阈值实时调整(0.1-0.9)
- **模型管理** - 动态加载和状态监控
- **详细日志** - 完整的操作记录系统
- **统计面板** - 实时显示检测统计信息

## 快速开始

### 环境要求

#### 硬件配置

- **处理器**: Intel Core i5 或同等性能以上
- **内存**: 8GB RAM (推荐16GB)
- **显卡**: 集成显卡即可 (NVIDIA GPU可选)
- **存储**: 2GB可用空间

#### 软件环境

- **操作系统**: Windows 10/11, Ubuntu 18.04+, macOS 10.14+
- **Python版本**: 3.8, 3.9, 3.10, 3.11
- **包管理**: pip 或 conda

### 安装方式

#### 方式一：源码运行

```bash
# 1. 创建虚拟环境
conda create -n ultralytics python=3.9
conda activate ultralytics

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python app.py
```

#### 方式二：可执行文件

1. **下载发布包** `FishDetection.zip`

2. **解压到任意文件夹**

3. **确保文件完整**：

   ```text
   FishDetection/
   ├── FishDetection.exe    # 主程序
   ├── best.pt             # AI模型文件
   └── _internal          # 依赖文件夹
   ```

   

4. **双击运行** `FishDetection.exe`

## 🎮 使用指南

### 首次运行

1. **启动应用** - 运行 `python app.py` 或双击可执行文件
2. **模型加载** - 系统自动加载 `best.pt` 模型文件
3. **目录创建** - 自动创建输出目录结构

### 操作流程

#### 图片检测模式

1. 点击"🖼️ 图片识别"进入图片检测页面
2. 点击"选择图片文件"选择单张或多张图片
3. 点击"开始检测"进行分析
4. 使用导航按钮切换图片，点击"保存结果"保存

#### 视频检测模式

1. 点击"🎥 视频识别"进入视频检测页面
2. 点击"导入视频"选择视频文件
3. 勾选"导出检测视频"可录制处理结果
4. 点击"开始分析"播放视频，使用"暂停"和"抓拍"控制

#### 实时摄像头模式

1. 点击"📹 摄像头识别"进入实时监控页面
2. 点击"启动摄像头"开始实时检测
3. 勾选"自动录制"可保存视频流
4. 使用"抓拍当前帧"保存精彩瞬间

#### 系统设置

1. 点击"⚙️ 系统设置"调整参数
2. 拖动滑块调节检测灵敏度
3. 点击"查看类别列表"查看支持的鱼类



## 常见问题

**Q: 模型加载失败怎么办？**
A: 确保`best.pt`文件存在于程序同目录下，且文件完整。

**Q: 摄像头无法启动？**
A: 检查摄像头权限，确保没有其他程序占用摄像头设备。

**Q: 检测速度慢？**
A: 尝试调高置信度阈值，或使用GPU版本加速。

**Q: 内存不足？**
A: 减少同时处理的文件数量，关闭其他占用内存的程序。
